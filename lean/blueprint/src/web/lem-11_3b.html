0.1.4 Lemma 11.3: Tail Symbol Properties

[Tail signature] PBP.tailSignature_D Per-cell contribution sum from tail cells in P’s column 0.

[Tail symbol] PBP.tailSymbol_D The symbol \(x_\tau \) in the last box of the tail: \(x_\tau := \mathcal{P}_{\tau _\mathbf {t}}(k, 1)\).

[Lemma 11.3(a): \(\varepsilon _\tau = 0\) iff \(x_\tau = d\)] PBP.tailSymbol_d_iff_d_in_col0 def:tailSymbol By layer monotonicity, \(d\) (layerOrd 4) can only appear at the bottom of column 0.

Proof ▼
If \(d\) exists in column 0, monotonicity forces it to the bottom. Conversely, bottom \(= d\) witnesses \(d\) in column 0.
Proof ▼

[Lemma 11.3(b): \(p_{\tau _\mathbf {t}} = 0\) iff \(x_\tau = s\)] def:tailSymbol, def:tailSignature (\(\Rightarrow \)) If \(x_\tau = s\) (layerOrd 1), all tail cells have layerOrd \(\leq 1\). Non-dot cells with layerOrd \(\leq 1\) are \(s\). So every tail cell is \(s\), contributing \(0\) to \(p\). (\(\Leftarrow \)) If \(x_\tau \neq s\), the bottom cell contributes \(\geq 1\) to \(p\).

Proof ▼
