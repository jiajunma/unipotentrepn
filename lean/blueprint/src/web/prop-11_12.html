0.1.10 Lemmas 11.9–11.11 and Proposition 11.12

[Lemma 11.11: No det twist] prop:11_8 There exist no \(\tau _1, \tau _2\) with \(\mathcal{L}_\tau [_1] \otimes (1,1) = \mathcal{L}_\tau [_2]\).

Proof ▼
prop:11_8 \(\otimes (1,1)\) negates the first entry: \((\mathcal{L}_\tau [_1] \otimes (1,1))^+ = 0\) always. So \(\mathcal{L}_\tau [_2]^+ = 0\), forcing \(x_{\tau _2} = s\) (Prop 11.8(b)). Similarly \(x_{\tau _1} = s\). But then \((\mathcal{L}_\tau [_1] \otimes (1,1))^- \neq 0\) while \(\mathcal{L}_\tau [_2]^- = 0\), contradiction.
Proof ▼

[Proposition 11.12: Injectivity modulo twist] lem:11_11, prop:11_8 If \(\mathcal{L}_\tau [_1] \otimes (\varepsilon _1, \varepsilon _1) = \mathcal{L}_\tau [_2] \otimes (\varepsilon _2, \varepsilon _2)\), then \(\varepsilon _1 = \varepsilon _2\), \(\varepsilon _{\tau _1} = \varepsilon _{\tau _2}\), and \(\mathcal{L}_\tau [_1] = \mathcal{L}_\tau [_2]\).

Proof ▼
lem:11_11, prop:11_8 \(\varepsilon _1 = \varepsilon _2\) from Lemma 11.11 (otherwise get \(\otimes (1,1)\)). Then \(\mathcal{L}_\tau [_1] = \mathcal{L}_\tau [_2]\), and \(\varepsilon _{\tau _1} = \varepsilon _{\tau _2}\) from the truncation pattern (Prop 11.8).
Proof ▼
