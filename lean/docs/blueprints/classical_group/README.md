# ClassicalGroup standalone task

This directory is the standalone Lean formalization task for the theorem in
`lean/docs/blueprints/classical_group/problem.md`.

It is intentionally separate from the `InducedOrbitToy` formalization.

Files:

- `problem.md`: copied problem statement.
- `blueprint.md`: copied verified proof blueprint.
- `formalization_plan.md`: Lean formalization plan and data-structure decisions.
- `Basic.lean`: initial standalone Lean module placeholder.
