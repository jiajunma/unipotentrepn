3.4 Lemma 11.3: Tail Properties

Definition 3.4.1 Tail signature

PBP.tailSignature_D def:PBP Sum of per-cell tailContrib from tail cells in P’s column 0.

Definition 3.4.2 Tail symbol

PBP.tailSymbol_D def:PBP Symbol \(x_\tau := \mathcal{P}_{\tau _\mathbf {t}}(k, 1)\) at the bottom of the tail.

Lemma 3.4.3 PBP signature sum, D type

PBP.signature_sum_D def:PBP, def:sign \(p_\tau + q_\tau = 2 \cdot |\tau |\) for D-type PBP.

Lemma 3.4.4 Tail signature sum identity

PBP.residual_identity_D def:tailSignature \(p_{\tau _t} + q_{\tau _t} = 2 c_1(O) - 2 c_2(O)\).
