Theorem 77 firstColSign diff invariant
✓
#

Uses

Lean declarations

  • AC.chain_firstColSign_eq_diff_sign

Let \(\mathcal{L}_\tau = \mathrm{AC.fold}\, \gamma \, (\mathrm{descent\ chain})\), with the descent chain producing signatures \(\mathrm{Sign}(\mathcal{L}_\tau )\) and \(\mathrm{Sign}(\mathcal{L}_{\tau ''})\) at successive levels. Then

\[ \mathrm{fcSign}(\mathcal{L}_\tau ) = \mathrm{Sign}(\mathcal{L}_\tau ) - \mathrm{Sign}(\mathcal{L}_{\tau ''}). \]

Equivalently:

\[ \mathrm{fcSign}(\mathcal{L}_\tau ) = (p_\tau - p_{\tau ''},\ q_\tau - q_{\tau ''}). \]
Proof ▶

By induction on the descent chain length. Base: \(\mathrm{fcSign}(\mathrm{AC.base}) = \mathrm{Sign}(\mathrm{AC.base})\) (AC.base_firstColSign_eq_sign). Step: Given \(\mathrm{fcSign}(\text{prev}) = \mathrm{Sign}(\text{prev}) - \mathrm{Sign}(\text{prev}')\), the post-twist and theta-lift formulas (Thm 69, Lem 65) give

\[ \mathrm{fcSign}(\text{curr}) = (\text{current } p - \mathrm{Sign}(\text{prev}).1,\ \text{current } q - \mathrm{Sign}(\text{prev}).2) \]

\(= (p_\tau - p_{\tau ''},\ q_\tau - q_{\tau ''})\) since the current parameters are \(\mathrm{Sign}(\tau ) = (p_\tau , q_\tau )\) and the previous source has \(\mathrm{Sign}= \mathrm{Sign}(\tau '')\) by IH.
