Definition 9Painted Bipartition

A painted bipartition (PBP) is \(\tau = (\mathcal{P}, \mathcal{Q}, \gamma )\) where \(\mathcal{P}\) and \(\mathcal{Q}\) are painted Young diagrams with root type \(\gamma \) satisfying:

  1. sym_P, sym_Q: \(\mathcal{P}.\mathrm{paint}(i, j)\) is in the allowed set for \(\mathcal{P}\) determined by \(\gamma \) (Def. 8), and likewise for \(\mathcal{Q}\).

  2. dot_match: \(\mathcal{P}(i,j) = \bullet \Leftrightarrow \mathcal{Q}(i,j) = \bullet \) (the outside-shape convention makes this an equivalence between cell memberships).

  3. row_s, row_r: each row contains at most one \(s\) across \(\mathcal{P} \cup \mathcal{Q}\); similarly at most one \(r\).

  4. col_c_P / col_c_Q: each column of \(\mathcal{P}\) has at most one \(c\); same for \(\mathcal{Q}\).

  5. col_d_P / col_d_Q: each column has at most one \(d\).

Definition 59Sign twist \(\otimes (\varepsilon ^+, \varepsilon ^-)\)

(Paper Eq. (9.15).) For \((\varepsilon ^+, \varepsilon ^-) \in \mathbb {Z}^2\) (in practice \(\{ \pm 1\} ^2\)), acts only on odd-length rows (\(\ell = i + 1\) odd, \(i\) even):

\[ \mathrm{twistBDRow}(i,\ \varepsilon ^+, \varepsilon ^-)(p, q) := \begin{cases} (p, q) & \ell \text{ even} \\ (\varepsilon ^{+\, h + 1} \varepsilon ^{-\, h}\, p,\ \varepsilon ^{-\, h + 1} \varepsilon ^{+\, h}\, q) & \ell \text{ odd, } h = i / 2. \end{cases} \]

Common cases:

  • \((1, -1)\): sign twist \(\otimes 1^{+-}\) (B/D post-twist when \(\varepsilon _\tau = 1\)).
