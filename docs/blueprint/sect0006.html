2.4 Painted Bipartition

Definition 9 Painted Bipartition
✓
#

Uses

  • Definition 5
  • Definition 8

Lean declarations

  • PBP

A painted bipartition (PBP) is \(\tau = (\mathcal{P}, \mathcal{Q}, \gamma )\) where \(\mathcal{P}\) and \(\mathcal{Q}\) are painted Young diagrams with root type \(\gamma \) satisfying:

  1. sym_P, sym_Q: \(\mathcal{P}.\mathrm{paint}(i, j)\) is in the allowed set for \(\mathcal{P}\) determined by \(\gamma \) (Def. 8), and likewise for \(\mathcal{Q}\).

  2. dot_match: \(\mathcal{P}(i,j) = \bullet \Leftrightarrow \mathcal{Q}(i,j) = \bullet \) (the outside-shape convention makes this an equivalence between cell memberships).

  3. row_s, row_r: each row contains at most one \(s\) across \(\mathcal{P} \cup \mathcal{Q}\); similarly at most one \(r\).

  4. col_c_P / col_c_Q: each column of \(\mathcal{P}\) has at most one \(c\); same for \(\mathcal{Q}\).

  5. col_d_P / col_d_Q: each column has at most one \(d\).

Remark 10
#

The notation \(\iota _\wp := \mathcal{P}\) and \(\jmath_ \wp := \mathcal{Q}\) appears in the paper and is used interchangeably. The index \(\wp \) is a primitive pair set which we usually omit, assuming the canonical \(\wp \).

Definition 11 PBPSet — PBPs of fixed shape
✓
#

Uses

  • Definition 9

Lean declarations

  • PBPSet

For \(\gamma \) a root type and \(\mu _P, \mu _Q\) Young diagrams,

\[ \mathrm{PBP}_\gamma (\mu _P, \mu _Q) := \{ \tau \in \mathrm{PBP} \mid \tau .\gamma = \gamma ,\ \tau .\mathcal{P}.\mathrm{shape} = \mu _P,\ \tau .\mathcal{Q}.\mathrm{shape} = \mu _Q\} . \]

Given a dual partition \(dp\), we set \(\mathrm{PBP}_\gamma (\check{\mathcal{O}}) := \mathrm{PBP}_\gamma (\mu _P(dp), \mu _Q(dp))\) where \(\mu _P, \mu _Q\) are derived from \(dp\).
