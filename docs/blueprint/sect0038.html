  • prop_11_9_PBP_D_numerical

Assume \(\tau _1, \tau _2 \in \mathrm{PBP}_D\) with \(h_{\mathrm{tail}}\) satisfied, \(x_{\tau _1} = d\), and the cross-twist first-component equation \(-p_{\tau _1, t} = p_{\tau _2, t} - 1\), plus \(x_{\tau _2} \neq s\): contradiction.

Proof ▶

\(x_{\tau _1} = d\) and \(x_{\tau _1} \neq s\) \(\Rightarrow p_{\tau _1, t} \geq 1\) (by Lem 87). So \(-p_{\tau _1, t} \leq -1\). Then \(p_{\tau _2, t} - 1 \leq -1\) \(\Rightarrow p_{\tau _2, t} \leq 0\). With \(p_{\tau _2, t} \geq 0\) (Lem 86), \(p_{\tau _2, t} = 0\). By Lem 87, \(p_{\tau _2, t} = 0 \iff x_{\tau _2} = s\), contradicting \(x_{\tau _2} \neq s\).

Lemma 109 Prop 11.11 — no det twist

  • BMSZ.no_det_twist_same_sign

There exist no \(\tau _1, \tau _2\) with \(\mathcal{L}_{\tau _1} \otimes (1, 1) = \mathcal{L}_{\tau _2}\) (the det twist would require impossible sign configurations).

Proof ▶

The \(\otimes (1, 1)\) twist negates the first entry: \((\mathcal{L}_{\tau _1} \otimes (1, 1))^+ = 0\) always. So \(\mathcal{L}_{\tau _2}^+ = 0\), forcing \(x_{\tau _2} = s\) (Prop 104(b)). But then \((\mathcal{L}_{\tau _1} \otimes (1, 1))^- \neq 0\) while \(\mathcal{L}_{\tau _2}^- = 0\): contradiction.

Proposition 110 Prop 11.12 — injectivity modulo \(\varepsilon \)

If \(\mathcal{L}_{\tau _1} \otimes (\varepsilon _1, \varepsilon _1) = \mathcal{L}_{\tau _2} \otimes (\varepsilon _2, \varepsilon _2)\), then \(\varepsilon _1 = \varepsilon _2\), \(\varepsilon _{\tau _1} = \varepsilon _{\tau _2}\), and \(\mathcal{L}_{\tau _1} = \mathcal{L}_{\tau _2}\).

Proof ▶

If \(\varepsilon _1 \neq \varepsilon _2\), the equation \(\mathcal{L}_{\tau _1} \otimes (1, 1) = \mathcal{L}_{\tau _2}\) arises (via \(\varepsilon _1 \oplus \varepsilon _2 = 1\)), forbidden by Lem 109. So \(\varepsilon _1 = \varepsilon _2\); cancellation gives \(\mathcal{L}_{\tau _1} = \mathcal{L}_{\tau _2}\). Then \(\varepsilon _{\tau _1} = \varepsilon _{\tau _2}\) follows from Prop 104’s truncation pattern.
