For any valid B-type dual partition \(dp\) with shape \((\mu _P, \mu _Q)\) derived from \(dp\):

\[ |\mathrm{PBP}_{B^+}(\mu _P, \mu _Q)| + |\mathrm{PBP}_{B^-}(\mu _P, \mu _Q)| = \mathrm{tripleSum}\big(\mathrm{countPBP\_ B}(dp)\big). \]
Proof ▶

Joint induction on the “Total”, “A1”, “A3” components following the B-type lifting structure (see CorrespondenceB.lean, main theorem card_B_combined). The balanced cases use Prop 45 and a fiber decomposition of \(\mathrm{PBP}_{B^\pm }\) along bottom-\(\mathcal{Q}\) paint. Primitive cases use Prop 44 and the M-type balanced bijection together with induction hypothesis on the shifted \(\mu _P\).

For any valid D-type dual partition \(dp\):

\[ |\mathrm{PBP}_D(\mu _P, \mu _Q)| = \mathrm{tripleSum}\big(\mathrm{countPBP\_ D}(dp)\big). \]
Proof ▶

By induction on \(dp\). Base \(dp = []\): both sides zero. Inductive step: split \(\mathrm{PBP}_D\) along the first-column structure; each sub-class (DD/RC/SS) has a bijection to a lower-level count via the double-descent recursion, which by IH equals \(\mathrm{tripleSum}(\mathrm{countPBP\_ D}(\mathrm{tail}(dp)))\).

\[ |\mathrm{PBP}_C(\mu _P, \mu _Q)| = \mathrm{countPBP\_ C}(dp). \]

For \(\mu _Q.\mathrm{colLen}(0) \geq \mu _P.\mathrm{colLen}(0)\): apply Prop. 42 to reduce to \(\mathrm{PBP}_D\); then invoke Prop. 55. For balanced: Prop. 43 gives \(\mathrm{DD} + \mathrm{RC}\) of the next-level D count.
