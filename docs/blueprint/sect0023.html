Four variants (paper Eq. (9.29)–(9.30)):

  • \(\hat\vartheta _{CD} : \mathbf{MYD}_C \to \mathbf{MYD}_D\) (single branch): \(\mathcal{E} \mapsto T^j(\mathrm{augment}((p, q), \mathcal{E}))\).

  • \(\hat\vartheta _{DC} : \mathbf{MYD}_D \to \mathbf{MYD}_C\) (two branches: standard with \(\mathrm{augment}((n, n), \mathcal{E})\); split with \(\mathrm{augment}((-1, -1), \Lambda (\mathcal{E}))\)).

  • \(\hat\vartheta _{BM}, \hat\vartheta _{MB}\) analogous.

Theorem 68 Theta lift signature preservation
✓
#

For each theta-lift direction with target \((p, q)\) satisfying the standard-case condition (\(p - \mathrm{Sign}(\mathcal{E}).1 - \mathrm{fcSign}(\mathcal{E}).2 \geq 0\), etc.): \(\mathrm{Sign}(\hat\vartheta (\mathcal{E})) = (p, q)\) (or \((n, n)\) in D\(\to \)C split).

Proof ▶

\(\mathrm{Sign}((p_0, q_0) :: \mathcal{E})\) evaluates signRow at the new index 0 (contributing \((|p_0|, |q_0|)\)) and shifts every existing row index by 1, which adds its \(\mathrm{firstColSignRow}\) contribution.

Theorem 69 Theta lift first-column signature
✓
#

Uses

  • ILS.thetaLift_CD_firstColSign

\(\mathrm{fcSign}(\hat\vartheta (\mathcal{E})) = (p - \mathrm{Sign}(\mathcal{E}).1,\ q - \mathrm{Sign}(\mathcal{E}).2)\) (the key firstColSign invariant).

Proof ▶

Each \(\hat\vartheta _\bullet \) is of the form \(T^j(\mathrm{augment}((p, q), \mathcal{E}))\) (or similar with \((n, n)\) or \((-1, -1)\)). Apply Lemma 66 then Lemma 64 for the \(T^j\) step.

Theorem 70 Theta lift first-entry formula

The first entry of \(\hat\vartheta (\mathcal{E})\) is \((p - \mathrm{Sign}(\mathcal{E}).1 - \mathrm{fcSign}(\mathcal{E}).2,\ q - \mathrm{Sign}(\mathcal{E}).2 - \mathrm{fcSign}(\mathcal{E}).1)\).

Proof ▶

thetaLift_*_firstColSign follows from Lemma 65 for the \(T^j\) step and the observation that \(\mathrm{augment}((p_0, q_0), \cdot )\) shifts every row’s first-column contribution.
