2.6 Main Counting Formulas (Propositions 10.11, 10.12)

These are the main counting theorems of [BMSZb] Section 10 — fully unconditional at the PBP level, depending only on shape/dp legality.

Proposition 2.6.1 Prop 10.11 B: counting for B type
✓
#

Uses

  • Corollary 2.5.3

Lean declarations

  • card_PBPSet_B_eq_tripleSum_countPBP_B
  • prop_10_11_B

For any valid B-type dual partition \(dp\) (sorted \(\geq \), even, positive) with shape match \((\mu _P, \mu _Q)\):

\[ |\mathrm{PBP}_{B^+}(\mu _P, \mu _Q)| + |\mathrm{PBP}_{B^-}(\mu _P, \mu _Q)| = \mathrm{tripleSum}\, (\mathrm{countPBP\_ B}\, dp). \]
