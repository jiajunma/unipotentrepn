2.1 PBP and Shape Conventions

Definition 2.1.1 Painted Bipartition (PBP)
✓
#

Lean declarations

  • PBP

A painted bipartition consists of two painted Young diagrams \(P, Q\) with paint symbols in \(\{ \bullet , s, r, c, d\} \), a root type \(\gamma \in \{ B^+, B^-, C, D, M\} \), and validity axioms (layer monotonicity, row/column constraints, dot match).

Definition 2.1.2 PBPSet — PBPs of fixed shape
✓
#

Uses

  • Definition 2.1.1

Lean declarations

  • PBPSet

For \(\gamma \in \{ B^+, B^-, C, D, M\} \) and Young diagrams \(\mu _P, \mu _Q\), \(\mathrm{PBP}_\gamma (\mu _P, \mu _Q) := \{ \tau : \text{PBP} \mid \tau .\gamma = \gamma , \tau .P.\text{shape} = \mu _P, \tau .Q.\text{shape} = \mu _Q\} \).
